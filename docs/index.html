Home

タブ UI を実装してみました。
要件は下記

  • not レスポンシブ
  • Internet Explorer: 9+
  • WAI-ARIA 対応で、キーボード操作が可能であること
  • PostCSS

なんとなく既存プロジェクトに導入できるような形にしてみたくて、割と現実的そうなブラウザ対応を取ることにしました。

WAI-ARIA は、やってみたかったのです。PC キーバインド設定するにあたっての JavaScript は Babel で実装できる環境まで作ったものの、結局 jQuery でササッと☺

PostCSS は、やりましょうという気運。実際使ったのは nested と simple-vars と calc.
