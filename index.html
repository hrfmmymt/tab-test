Home

タブ UI を実装してみました。
要件は下記

  • not レスポンシブ
  • Internet Explorer: 9+
  • WAI-ARIA 対応で、キーボード操作が可能であること
  • PostCSS

not レスポンシブなのはスマホでタブ切り替えの UI を実装するのはあまり気乗りしないためです。

transform, transition をなるべく試していきたいんですけど、割と現実的なブラウザ対応を取ることにしました。

WAI-ARIA はただただやってみたかったのです。JavaScript は Babel で実装できる環境を作りましたが、結局 jQuery でササッと。

PostCSS はやりましょうという気運。実際使ったのは nested と simple-vars
